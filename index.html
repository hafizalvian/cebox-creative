Panduan Penggunaan Framework Cebox

1. Cara Instalasi

Tambahkan file CSS berikut ke dalam bagian head HTML Anda:

• cebox.css - Untuk styling dasar

• button.css - Untuk komponen tombol

• tables.css - Untuk komponen tabel

2. Komponen Tombol

Framework ini menyediakan 3 jenis tombol:

  • Tombol Utama: Gunakan kelas "cebox-btn cebox-primary"
  • Tombol Sukses: Gunakan kelas "cebox-btn cebox-success"
  • Tombol Outline: Gunakan kelas "cebox-btn cebox-outline"

3. Komponen Tabel

Buat tabel responsif dengan menggunakan kelas "cebox-table". Tabel akan menyesuaikan secara otomatis untuk tampilan mobile.

4. Kelas Layout

  • container: Pembungkus konten utama
  • nav-container: Pembungkus navigasi
  • footer-content: Pembungkus bagian footer

5. Desain Responsif

Framework ini menggunakan pendekatan mobile-first dengan breakpoint:

  • Mobile: Hingga 768px
  • Tablet: 768px hingga 992px
  • Desktop: 992px ke atas
